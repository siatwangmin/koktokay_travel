# 可可托海之家庭度假型滑雪攻略
首先说明一下，这个攻略是针对特定家庭度假型客户去可可托海滑雪度假的攻 略，不是面向那些疯狂爱好者或专业滑雪者所用。

我去过不少雪场滑过雪，瑞士、日本、德国、芬兰，也包括国内的崇礼、北大 壶、松花湖，这次圣诞假陪娃本来打算去加拿大惠斯勒滑雪，结果加拿大签证 从暑假办到年底都不出签（现在也没有出），只好临时转战可可托海。 但真的没有想到可可托海的雪场是如此的好，超过我过去去过的任何雪场。

可可托海的雪场有几个非常好的特点：

1、雪好，都是自然雪，滑到下午结束时雪质量依然不错；

2、人少，缆车站几乎从来不用排队，甚至经常有雪道包场的感觉；

3、出太阳的日子很多，温度高，体感舒适。对我来说，这点太重要了。这次七天的滑程，六天是晴天，温度基本上-8 - -10度左右，实在是太舒服。都说日本雪好，但其实一个很大困扰是几乎天天阴天，能见度低，其实挺没有意思；

4、可用于滑雪的时间也很长，基本上可以做到上午十点开滑，下午六点结 束，远长于一般内地和日本雪场。你用滑呗去记录，发现自己会比一般雪场滑 的里程多不少；

5、有些特别吸引人的附加价值，比如夕阳，在一般雪场是很难看到夕阳的， 但在可可托海，几乎每个晴天都可以看到极为美丽的高山夕阳，非常漂亮，单单这个，感觉就值回票价；比如网红小狐狸，在红缆山顶的附近几乎每天都可 以看到一只很靓很可爱的小狐狸；道外的野雪，尤其是小狼，滑上一趟，难度不算大，乐趣无穷；

![v2-a4e161c7e70a4bc07e0f11968409a3c4_1440w](https://github.com/user-attachments/assets/df62b5bb-5ad2-4150-be50-df64220cf3c7)

但但但，这么好的雪场，其实吸引的大部分都是狂热的滑雪爱好者，家庭度假型的客户极少。从我自身经历来看，原因是以下几个：

1、长途交通不便利。去可可托海必须经过乌鲁木齐转机，而航班不衔接，比如深圳去可可托海，当天15:25分就到了乌鲁木齐，但不好意思，没有衔接的航班，必须第二天早上再坐飞机去可可托海，而可可托海机场到雪场还要两个小时，这就要两天，可谓长途慢慢。家庭滑雪度假一般就一周左右（这已经算长的），这个路途时间就太长了。回程机票倒是衔接的，但很不幸的事情是，临飞的前一个晚上告诉我可可托海到乌鲁木齐的飞机取消了！一下人就傻眼了。

2、来回雪场交通不便利。大部分雪友住在可可托海镇，每天来回雪场的时间要两小时，坐摆渡大巴，而且人多的时候还容易赶不上。下大雪的时候，还可能道路封堵，上不了山去不了雪场。

3、除了飞机和摆渡车，其他租车服务基本上都是雪友群里口口相传的黑车服务。我这次向某个司机交了200定金，结果就被人放了鸽子，连投诉都不知道找谁。

4、配套设施少，服务态度有待于改善，度假氛围几无。相比于松花湖、北大壶、崇礼的雪场，配套设施和服务都差的相当远，似乎除了滑雪就没有别的事可做。服务上需要改进的地方也很多，比如缆车十点开始运行，但售票也是十点才开始，这就有点匪夷所思，类似这样的服务小细节需要改善的很多。也没有温泉，餐饮选择也少，真正的新疆菜其实吃不太惯。酒店水准普遍还处于较低水准，连锁酒店品牌很少。镇上点外卖没有美团，只有饿了吗，而且选择少、价格贵、配送慢。习惯内地大城市或者松花湖那样上佳配套（甚至包括音乐会、电影院和奢侈品店）的雪场，可可托海滑雪其实相当是缺乏享受型度假设施和感受的。

5、对新疆安全的担心，很多人还是印象中十年前的新疆，有暴乱分子，到处是武警，菜刀上锁链。对外籍客人也非常不友好，雪场酒店就只能接待大陆旅客，我的孩子是香港籍贯，理论上是不能接待的。

相信上面的几点足以吓退绝大部分滑雪度假家庭。因为，家庭度假型客户和一般狂热的滑雪爱好者最大的不同是滑雪是承载家庭欢乐的载体，是过程而不是目标，他们需要享受、需要便捷、需要确定性，他们支付能力也能强，愿意花钱买享受，但前提是要能买到享受。

可可托海就像一个没有打磨的宝石，非常美但很粗粝，那么度假型客人怎么才能享受美好的雪场、雪景，享受美好的度假生涯，而尽可能减少交通、服务等短板的困扰呢？

按句时髦的话来说，就是习惯南方细糠的友友们如何食粗糙但其实美味的西北粗粮？

以下就是一些非常实用型的攻略：

一、长途交通以一天到达为基本原则

度假型客人的时间非常宝贵。无论来回，应该以一天到达目的地为基本的原则。两天实在是太长了。

以深圳客人为例，去的时候，不要去做深圳-乌鲁木齐-可可托海的联程，应该选择早点到乌鲁木齐的航班，然后包车或拼车去可可托海镇。我就是这样操作的，下午3点半到乌鲁木齐，大致4点半从机场出发，晚上10:30-11:00之间到达可可托海镇，其实并不觉得累，反而有种兴奋的感觉。新疆其实和我们内地是有时差，10点钟相当于国内的8点，这个点其实并不晚。而且一路和司机聊聊新疆的风土人情、民族融合、治安形势等，非常有意思，再看看手机、打打瞌睡，时间过得很快。到镇上，并不是雪场，第二天转乘酒店门口的摆渡大巴上雪场就可以了，第二天就是一个完整的滑雪Day。

从可可托海回程应考虑可可托海-乌鲁木齐-深圳的联程，这是一天能到的。当然像碰到我这次可可托海-乌鲁木齐航班取消，就要及时寻找包车，确保一天的回程时间。我们最后一天的滑行结束后，要坐摆渡大巴到可可托海小镇，这样第二天无论是坐飞机还是包车去乌鲁木齐都是非常方便。

市场上并没有包车的公开信息，可以在雪友群里去找，有很多的资源。还有一种方法，这是酒店店家告诉我的，酒店前台有一张很多司机（全部是少数民族）的电话，随便打通一个司机，说明你们的要求，那个司机群是完全互通的，一点接触，全面服务，很快他们就能安排好。这次回程时可可托海到乌鲁木齐航班取消后，我前一天晚上就是这样，结果是非常好，每人260，八人坐的高端商务小巴士，早上酒店接我们，非常舒适就顺利到了乌鲁木齐机场。我把这个司机们电话发到这里，大家可以试试。

二、住宿要选雪场酒店

可可托海雪场从去年开始，有了雪场酒店。住在雪场酒店，优势太明显了，两个小时的来回大巴时间省去了，早上可以滑顶门头道雪，晚上看夕阳，滑到多晚都不怕，中午困了还可以回酒店小睡一会。

对于可可托海滑雪来说，不住雪场酒店就不是度假！

可可托海雪场酒店有两个，一个叫芳华可可，在雪具大厅的附近，一个叫西街48吧，位于中间的雪道边。芳华可可价格从1080（标间）起，这次我们前几天住的是1280的更高档房型（豪华标间），感觉非常大，打个地铺再住1-2人没有问题（懂的都知道我说什么），后两天因为芳华可可无房，换到西街48。西街48是loft房型，价格是1480，绝对的滑入滑出，早餐要滑下去吃，居住感受还不如芳华可可。所以，我的建议是尽量住芳华可可。

虽然服务细节上还存在诸多不如意，但整体素质还是不错了，大致4星。

酒店怎么定呢？两种方法，一种是上“大喀”小程序（新疆滑雪的官方微信号）订，第二种是直接找酒店前台订，电话微信同号19990659975，前台房源更多更可靠一些。

雪场酒店退起来非常方便（似乎可可托海镇上酒店都是这样），提前3天退都能全款回，所以呢，可以早下手。

从缆车上看，可可托海雪场的酒店还在建设中，规模和类型都在扩大，相信明年会有更多选择，也会有很多的配套服务（比如温泉），对于度假型雪友更会友好。明天会更好！

至于说雪场酒店不接待非大陆客户，这个不用理，不登记不就可以了吗？

这里要特别说一下，第一天到达晚上和滑雪的最后一个晚上（要坐第二天上午的飞机），要住在可可托海小镇上，而这里选择就多了。但请相信一分钱一分货，贵一定是有贵的道理，便宜就一定有它的问题。第一晚是芳华故事里，可可可能最有名的老牌酒店，翻新后感觉挺好，三人间598，而走之前那个晚上住在雪域可可酒店，228的双人房，一下就差了很多，只能说勉强可住。

三、餐饮怎么选？

在雪场住，那就只能在雪场的餐馆消费。

早餐雪场酒店会提供，在雪具大厅的三楼，质量还行，标准四星级酒店自助早餐。

对于一般雪友，雪场的餐饮集中在雪具大厅的二楼和三楼，二楼是大排档类型，感觉和北京南山滑雪场档次类似，明显差于松花湖。三楼就是酒店早餐所在地，提供各类炒菜，吃过一次，感觉价格很高，味道一般。可能是处于垄断位置的餐饮，生意不用愁，就没有花心思去改进餐饮质量。

不过，有三个雪场餐馆让人印象深刻，值得推荐。

第一个是灰缆栈顶海拔2981米的诺茉西餐厅，这是一家西餐厅，环境极佳，意大利面、披萨、牛排是主打，味道相当地道，尤其是意大利面惊到了我们。晚餐的时候正是最美的夕阳时光。而且还有一个特别的小彩蛋，晚餐的客户，有个加班的缆车会吧客人运到灰缆山下站，然后无缝衔接夜滑路线。吃顿美味的西式晚餐，送夕阳美景，送缆车专车，再送一趟夜滑，实在一个值得珍惜的宝藏餐馆。



![v2-a02abac394f8d9acd03690ec915ed6e9_1440w](https://github.com/user-attachments/assets/e9e8e195-0076-4d93-a4a4-196211d5f56e)





如果不要坐加班缆车，也可以自己往下滑，当时美景真是让我和我儿子惊呆了，一轮巨大的明月就挂在雪道后面，我们在清冷的明月前滑雪，那个感觉简直是无以伦比。天啊，太美了。手机没有办法拍出来，后悔没有带长焦相机过来。

第二个是西街48一楼的湘菜馆，就在灰缆下站的上面。湘菜非常地道，尤其白辣椒炒肉，深得湘菜的精华。很难想象，在遥远新疆可可托海滑雪道边，有家如此地道的湘菜馆。

第三个是小狼野雪滑行终点处，也就是雪具大厅往1900站滑雪雪道的中部，有个两个小餐馆，一是日式，一是韩式，也是非常有味道，也符合年轻人的口味。估计这个只能午餐了，晚餐的话除非老板愿意开车送大家会雪具大厅才行。

这三家餐馆除了贵，没有毛病。对于价格，要有心理准备。大致按欧洲的来就可以了。但是相比于昂贵的教练费用、住宿机票费用，再吃能吃多少？抱着平常心就好。

相比于那些配套完善的雪场，我建议还是多带些水果，带些方便面、方便食品，带多点零食。毕竟这个地方供应条件很差。尤其是水果，新疆餐饮青菜少而贵，水果补充维生素以及带来愉悦的感受，我这次带了一箱粑粑柑，正好吃完，感觉非常好。

四、雪具怎么办？

相比其他雪场，到达可可托海雪场要折腾太多了，雪具如果随身带，那真是累死人，而且可能车就运不了。那怎么办呢？

这个时候，德邦雪具达就有奇效了。我在深圳用德邦雪具达（微信搜一下）提前一周寄到雪场酒店，让放到前台，然后滑雪的当天在前台拿到雪具行李，直接就可以开滑，结束的时候，打好包，让德邦小哥寄回深圳就可以。实在是太便利了。来回550左右，省去了无数的麻烦，实在是值得推荐。

按照这个思路，比如大箱的水果、补给，比如在酒店打地铺的装备，都是可以这样邮寄过去的。新疆远，留够足够的时间就好了。

我也同时，非常建议大家买上insta 360滑雪拍摄神器，新疆雪场人少雪好阳光多，实在是拍摄滑雪美姿的绝佳之地。无人机也很值得带，夕阳美景下无人机拍摄也很惊艳。我下次再去的时候，还带个长焦的佳能相机，很想拍一下大大月亮背景下滑雪的美妙姿态。

另外，我这次还专门从淘宝买了折叠易携带的洗脚盆，效果非常好，每天滑雪后泡一泡，实在是巴适的狠，非常值得推荐。当时甚至准备买个便携的泡澡浴缸，箱子也装的上，但时间来不及了。有兴趣可以大胆试试。

还有雪票说一下，可可托海的雪票价不高，相比北大湖、松花湖便宜的多。般都是现场买，似乎没有别的通道薅羊毛。其实是有的，就是在抖音商城上可以找到3日票和7日票，那还是能省不少钱的。学生就买学生票，能便宜不少。

最后说一点，关于野雪滑行。稍微搜一下攻略，大狼小狼是很知名的。

首先大家还是要安全为第一，水平不够不要轻易去尝试，水平不错也一定要跟教练或高手一起去，小狼难度较低，较为安全，也很有趣，值得尝试。而大狼，不是高手不要去，很多蘑菇，难度太高了。

结束我的攻略介绍。希望对大家有所帮助，让度假型客户想去可可托海、敢去可可托海，并享受可可托海的滑雪行程。随着大量配套酒店的兴起，相信可可托海明年会更好。
